\section{Competency Questions}

\begin{enumerate}
% For 1st point
    \item What is the shortest collaboration path between two actors or other persons involved in audiovisual productions?
    \item Through which creative works are two persons connected?
    \item Which persons are central in the collaboration graph?
% For 2nd point
    \item Which local entities can be aligned with external resources such as Wikidata?
    \item What additional contextual information can be added through external enrichment?
% For 3rd point
    \item Which episodes belong to a given series?
    \item Which genres, languages, and regions are associated with a given creative work?
    \item Which related entities can be discovered when navigating from a film, person, or series in the knowledge graph?
\end{enumerate}
