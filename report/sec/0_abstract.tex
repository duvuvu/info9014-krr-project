\begin{abstract}
This report covers Milestones 2 and 3 of the CineExplorer project. In Milestone 2, an ontology is designed to capture the main concepts of the audiovisual domain and their relationships. The ontology is guided by competency questions and implemented in OWL 2 DL, serialized in Turtle syntax, and edited in Protégé. Key design decisions include the introduction of \texttt{CreativeWork} as a central abstraction, the use of a reified \texttt{Participation} class to represent contextual relations between persons and works, and the use of inverse properties to support graph traversal.

In Milestone 3, the relational database is annotated using an R2RML mapping that generates an RDF knowledge graph conforming to the ontology. The mapping covers the main entity types and applies a stable IRI strategy based on IMDb identifiers.
\end{abstract}
