\begin{abstract}
This report describes CineExplorer, a knowledge-graph application built over the official \ac{imdb} non-commercial dataset and deployed for \ac{sparql} querying and Linked Data browsing. The system covers the full Milestone~1--4 pipeline: a relational schema reduced from a 14-relation \ac{erd}; a 1{,}611{,}676-triple knowledge graph generated by an \ac{r2rml} mapping over the migrated database; an \ac{owl}~2 \ac{dl} ontology centred on \texttt{CreativeWork} and a reified \texttt{Participation} class to capture per-credit context; and an Apache Fuseki deployment paired with the Brwsr Linked Data browser. Two non-trivial demonstrators close the loop. A Bacon-number analysis using \ac{sparql} property paths reaches 92\% of all 38{,}067 persons within three collaboration hops of Kevin Bacon, motivating the project's earlier migration from a 174-title curated sample. A \ac{shacl} structural validation contrasts closed-world constraint checking with the open-world semantics of the \ac{owl} ontology, and surfaces a real 25-title mapping gap that the open-world view would silently allow. Together the artefacts demonstrate end-to-end use of database design, ontology engineering, \ac{r2rml} mapping, \ac{sparql} federation, and constraint-based \ac{kg} validation.
\end{abstract}
