\begin{abstract}
In Milestone 2, we design an ontology that captures the main concepts of the audiovisual domain and their relationships. The ontology is guided by clearly defined competency questions and is implemented in OWL 2 DL using Turtle and Protégé. Key design decisions include the introduction of \texttt{CreativeWork} as a central abstraction and the use of an explicit \texttt{Participation} class to model contextual relations between persons and works. The resulting ontology provides a structured and extensible foundation for the next milestones.
